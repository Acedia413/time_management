# О проекте (ВКР)

В рамках преддипломной практики я разрабатываю информационную систему. Проект состоит из двух основных частей: клиентской (frontend) и серверной (backend).

## Технологический стек

### Frontend

Использую **Next.js 16** с App Router. Это позволяет мне удобно организовать маршрутизацию и использовать возможности React Server Components.

*   **Язык**: TypeScript.
*   **Стили**: Tailwind CSS 4. Решила использовать его для быстрой верстки, основные стили вынесла в `globals.css`.
*   **Сборка**: стандартная через `npm run build`.

### Backend

Серверную часть пишу на **NestJS 11**.

*   **Архитектура**: Модульная (стандартная для NestJS).
*   **База данных**: PostgreSQL 15. В качестве ORM выбрала **Prisma**, так как она обеспечивает строгую типизацию.
*   **API**: REST.

### Инфраструктура

Для удобства разработки базу данных поднимаю в Docker контейнере.

*   `docker-compose.yml` — конфигурация для запуска PostgreSQL.
*   Данные сохраняются в volume `postgres_data`, чтобы не терялись при перезапуске контейнеров.

## Запуск проекта

Для запуска требуется Node.js и Docker.

1.  Поднять базу данных: `docker-compose up -d`
2.  Запустить бэкенд: `cd backend && npm run start:dev`
3.  Запустить фронтенд: `cd frontend && npm run dev`
