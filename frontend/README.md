# Фронтенд часть ВКР

Это клиентская часть моего приложения. Разрабатываю на **Next.js** (версия 16) с использованием **React 19**.

## Стек технологий

*   **Фреймворк**: Next.js (App Router)
*   **Язык**: TypeScript
*   **Стилизация**: Tailwind CSS 4 (настроена через CSS переменные в `globals.css`)
*   **Шрифты**: Geist (оптимизированы через `next/font`)

## Как запустить

1.  Установить зависимости:

    ```bash
    npm install
    ```

2.  Запустить сервер разработки:

    ```bash
    npm run dev
    ```

3.  Открыть [http://localhost:3000](http://localhost:3000) в браузере.

## Структура

*   `src/app` — страницы и роутинг приложения.
*   `src/components` — переиспользуемые компоненты (Header, Sidebar, Dashboard и др.).
