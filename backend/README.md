# Бэкенд часть ВКР

Серверная часть моего приложения. Реализована на фреймворке **NestJS**.

## Стек технологий

*   **Фреймворк**: NestJS 11
*   **Язык**: TypeScript
*   **База данных**: PostgreSQL (работа через Prisma ORM)
*   **Тестирование**: Jest

## Как запустить

1.  Установить зависимости:

    ```bash
    npm install
    ```

2.  Запустить в режиме разработки (с отслеживанием изменений):

    ```bash
    npm run start:dev
    ```

## База данных

Для работы нужна запущенная база данных PostgreSQL. Я использую Docker Compose для локального запуска.

```bash
docker-compose up -d
```
